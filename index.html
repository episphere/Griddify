Completing Core Segmentation

Begin by fine-tuning the segmentation parameters to ensure the algorithm detects most of the cores. Make any manual adjustments if needed to capture all cores accurately. Once you're satisfied with the segmentation, click the "Finalize Segmentation" button. You can then proceed to the Gridding tab to continue.

Interactive Core Editing

To add a core, simply click on the desired location within the image. If you wish to remove a core, hold the Shift key and click on the core you want to remove.

Gridding Instructions

First, ensure that the image is rotated correctly. If not, adjust the image rotation parameter. To adjust the size of each core, hold the Alt key and move the cursor. To edit the properties of a core, double-click on the core to open the edit menu.
